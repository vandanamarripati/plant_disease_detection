About LeafScan

An end-to-end demo of an AI-assisted plant disease detection tool, built with a Flask backend, a MobileNetV2-based CNN, and a live weather advisory layer.

The model

A transfer-learning CNN built on MobileNetV2, trained on the 38-class PlantVillage dataset covering 14 crop species. See model/train_model.py to train it yourself.

Demo mode

Without a trained model file present, the app automatically falls back to a heuristic color-based classifier so every page stays fully interactive out of the box.

Weather advisories

Live data from the OpenWeatherMap API is converted into rule-based farming tips — spray timing, irrigation, and disease-risk alerts.
